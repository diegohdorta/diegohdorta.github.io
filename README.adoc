== Building the Page

. Get the prerequisites to build the AsciiDoc page on host machine:
+
[%autofit]
----
$ apt install ruby
$ gem install bundler
----
+
. To build and generate a local page:
+
[%autofit]
----
$ bundle install
$ bundle exec jekyll serve
----
